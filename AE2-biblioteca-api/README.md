# 📚 Biblioteca API — CRUD REST con Express

API REST desarrollada con **Node.js y Express** para gestionar un catálogo de libros almacenado en un archivo JSON local.

Este proyecto implementa operaciones **CRUD (Create, Read, Update, Delete)** y utiliza el módulo **File System** de Node.js para persistir los datos en un archivo `catalogo.json`.

---

# 🚀 Tecnologías utilizadas

- Node.js
- Express
- File System (`fs/promises`)
- Nodemon (para desarrollo)
