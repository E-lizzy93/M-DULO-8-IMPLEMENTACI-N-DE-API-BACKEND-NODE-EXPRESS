Estructura del Proyecto:

catalogo.json: Base de datos local en formato JSON que almacena la información de los libros (títulos, autores, géneros, etc.).

index.js: Punto de entrada de la aplicación donde se configuran los middlewares y se definen los métodos HTTP (GET, POST, PUT, DELETE) para el manejo de la biblioteca.

package.json / package-lock.json: Archivos de configuración de NPM con las dependencias del proyecto (Express).

README.md: Documentación con la descripción del CRUD y las instrucciones para probar los diferentes endpoints.
