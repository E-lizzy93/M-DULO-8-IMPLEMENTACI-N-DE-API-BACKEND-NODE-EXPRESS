Descripción de la Entrega:
En esta primera etapa del módulo, se ha establecido la base del proyecto enfocada en las mejores prácticas de seguridad para entornos Node.js.

Estructura y Avances:

Gestión de Variables de Entorno: Implementación de archivos .env para proteger credenciales sensibles (DB, Secret Keys), asegurando que no sean rastreadas por el control de versiones mediante el uso de .gitignore.

Servidor Base con Express: Configuración de un servidor robusto preparado para la implementación de Middlewares de seguridad.

Estructura de Carpetas Profesional: Organización orientada a la escalabilidad, separando rutas, controladores y modelos.

Configuración de Dependencias: Gestión de paquetes esenciales para el manejo de seguridad y autenticación que se desarrollarán durante el módulo.
